This software uses a Fibonacci Left Shift Register to encrypt and decrypt images using a 16 bit string.

The included png's "encode" and "decode" show an example of the encryption and decryption process.

/**********************************************************************
 *  Briefly explain the workings of the features you implemented.
 **********************************************************************/

1. Go to this folder through the linux console
2. Type "make"
3. Type "./PhotoMagic [Input Image file name] [Desired Output image file name] [16 Character String consisting of 0's and 1's]
4. Access the output image in the folder
5. You can reenter the output image into the input image with the same string to decrypt the image
